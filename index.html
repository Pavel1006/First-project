How websites works ?

All web sites at the most basic are just a panch of files that are storedin your computer. Computer is called a server. Server is conected to the internet . You can load the website through a borwser like: Google Chrome, FireFox, Safari etc. For a website to work well it needs front end and back end

Front end

Front end is acually that we can see on our browser. The frontend of a website is made up of 3 types of file HTML,CSS and JavaScript

HTML

HTML is the base of web sites. With it we can put the iformation on our website. It works with tags. It is a text editor.

CSS

With CSS we can arrange certain things, we can give them colors and backgrounds, etc. It too use tags.

JavaScript

It is a laguage programating. It can make it respond to some user data. It can make buttons with different functions etc.

Aditional front end could be: SASS for CSS , JavaScript framework,

Back End

Back End is the part of web site that we can't relly see but it heandles a lot of the logical and functionality that is necessary for everything to work. The basic Back Ends are server, programing language and database.

Server

Server is the computer where all websites files and database are stored. The traditional server is windows. Windows are considered centralized.

Programing Language

Programing Language is used to write the function and logic of your app. Populare programming language are: Ruby, PHP, Java etc.

Database

Database are where u stored your information on your server for your website.

IDE or Code Editor is an app that we can edit the source code. The most popular Code editor are Visial Studio, Atom, Notepad++ etc.
